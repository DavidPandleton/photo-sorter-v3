# Keyboard Shortcuts Reference

## Rating & Navigation

| Key | Action |
|------|--------|
| `1` | Rate BAD |
| `2` | Rate OK |
| `3` | Rate GOOD |
| `0` | Unrate |
| `N` | Next image |
| `P` | Previous image |
| `Home` | Jump to first image |
| `End` | Jump to last image |
| `U` | Toggle filter unrated only |
| `Enter` | Finish sorting & export |
| `Escape` | Return to main menu |

## Advanced Rating

| Key | Action |
|------|--------|
| `Space` | Toggle pick/flag (gold star) |
| `Ctrl+1` | Star rating: 1 star |
| `Ctrl+2` | Star rating: 2 stars |
| `Ctrl+3` | Star rating: 3 stars |
| `Ctrl+4` | Star rating: 4 stars |
| `Ctrl+5` | Star rating: 5 stars |
| `Ctrl+Z` | Undo last rating |

## Display

| Key | Action |
|------|--------|
| `I` | Toggle info panel (progress, filename, rating, EXIF) |
| `C` | Toggle compare mode (side-by-side) |
| `F` | Toggle fullscreen |
| `H` | Toggle HUD (controls legend) |

## Zoom

| Key | Action |
|------|--------|
| `Ctrl+`+ | Zoom in |
| `Ctrl+-` | Zoom out |
| `Ctrl+0` | Reset zoom (fit to window) |

## Navigation

| Key | Action |
|------|--------|
| `Ctrl+G` | Jump to image number |
| `Delete` / `Backspace` | Permanently delete current image |
| `R` | Rotate right 90° |
| `Shift+R` | Rotate left 90° |

## Directory

| Key | Action |
|------|--------|
| `Ctrl+O` | Open folder |
| `Ctrl+Q` | Quit |

## Gamepad Controls

| Button | Action |
|--------|--------|
| **A (Cross)** | Rate GOOD |
| **X (Square)** | Rate OK |
| **B (Circle)** | Rate BAD |
| **LB / RB** | Previous / Next image |
| **L-Stick** | Pan image |
| **R-Stick** | Zoom in/out |
| **LT / RT** (L2/R2) | Rotate left / right |
| **Start** | Finish sorting & export |
| **Select/Back** | Return to menu |
| **Y (Triangle)** | Reset zoom |
| **R-Thumb** | Toggle HUD |
