WorkflowCara pakai

Three keys. One export. Tiga tombol. Satu export.

Open a folder, press 1/2/3 on every photo, then hit Enter. Your files are sorted into folders. That is it. Buka folder, pencet 1/2/3 di tiap foto, tekan Enter. File lo ke-sortir otomatis. Sesimpel itu.

1

Open any photo folder Buka folder foto

Point Photo Sorter to a directory of photos. RAW files from your camera, JPEGs from your phone, anything. It walks the folder and builds a filmstrip instantly. Arahin Photo Sorter ke folder foto lo. Mau file RAW dari kamera, JPEG dari HP, apa aja jalan. Aplikasi bakal scan folder dan bikin filmstrip langsung.

2

Rate with your keyboard Rating pake keyboard

1 for bad, 2 for maybe, 3 for good. Or use star ratings with Ctrl+1-5. All without touching a mouse. 1 buat jelek, 2 buat ragu, 3 buat bagus. Atau pake rating bintang Ctrl+1-5. Semua tanpa mouse.

3

Export and done Export, beres

Press Enter. Files are moved into BAD/, OK/, and GOOD/ folders. One-click restore if something went wrong. Tekan Enter. File pindah ke folder BAD/, OK/, dan GOOD/. Kalo salah, tinggal restore satu klik.

FeaturesFitur

Built for culling. Nothing else. Dibuat khusus buat milah foto. Nggak ada yang lain.

Photo Sorter does one thing — let you sort photos fast. No editing, no cataloging, no library management. Photo Sorter cuma ngelakuin satu hal — milah foto dengan cepet. Bukan buat edit, katalog, atau manajemen library.

RAW previews at full speed Preview RAW tanpa nunggu

Reads embedded JPEG previews from NEF, CR2, CR3, ARW, DNG, ORF, RW2, PEF files. No waiting for full RAW decode. Zero-AI focus meter detects blur instantly. Baca preview JPEG dari file NEF, CR2, CR3, ARW, DNG, ORF, RW2, PEF. Nggak perlu nunggu decode RAW penuh. Deteksi blur otomatis langsung kelihatan.

Compare mode Mode compare

Press C to split the screen and compare two photos side by side. Pick the sharper one instantly. Tekan C buat liat dua foto berdampingan. Langsung tau mana yang paling tajam.

EXIF at a glance Info EXIF langsung liat

ISO, aperture, shutter speed, lens, camera model. Extracted from RAW and JPEG files and cached in SQLite. ISO, aperture, shutter speed, lens, tipe kamera. Diambil dari file RAW dan JPEG, disimpan di database.

Gamepad support Bisa pake controller

Plug in an Xbox or PlayStation controller. USB or Bluetooth. Rate, navigate, zoom, pan — all without a keyboard. Colok controller Xbox/PlayStation lewat USB atau Bluetooth. Rating, navigasi, zoom — semua tanpa keyboard.

Checkpoint safety Aman dengan checkpoint

Before every export, Photo Sorter saves a checkpoint. If files go to the wrong folder, one click restores everything. Sebelum export, Photo Sorter nyatetin checkpoint. Kalo file salah pindah, satu klik balik lagi ke posisi semula.

SQLite persistence Data aman terus

Ratings, star ratings, picks, rotation — everything survives restart. Open the same folder tomorrow and pick up where you left off. Rating, bintang, pick, rotasi — semua aman meskipun aplikasi ditutup. Besok buka lagi, lanjut dari tempat terakhir.
